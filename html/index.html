This app helps users to keep track of their daily nutrition intake based on a default 2000 calories diet.
Users will get a picture of a nutrition label package indicated on goods they bought from the folder.
Then, the nutrition values and breakdown from the label will stored onto a firestore database
The app will pull the data from the firestore database and display it in the piecharts.
The piecharts will show the amount of intake when selected on the screen as well as how much left is needed to reach the daily goal.
Chart drawn on the app comes in six parts, showing each of the following nutritional values.
  • Cholesterol
  • Saturated Fat
  • Total Fat
  • Dietary Fiber
  • Total Carbohydrate
  • Sodium
Each of the chart displays in different color indicator.
  • Grey means zero intake.
  • Colored means there are some intake. There are two different color, green and blue base color.
    Each of the color has lighter color and darker color version of it. The lighter color means the users' input values for the day, while the darker color means the daily recommended intake.
  • Red means the user has exceeded the daily intake value.
The user can modify their information stored in the database through the settings page.
The settings page has five buttons:
  • Change username: To change the users' name displayed on the header of the dashboard.
  • Change password: To change the users' old password to a new password. Minimum of length 6*.
  • Clear Data: To clear the users' current day of daily intake of nutrients in the database.
  • Return to Dashboard: Go back to the dashboard where the piecharts are displayed.
  • Logout: Log the user out from the app.
Currently the app only supports goods or foods with nutrition label packaging.

*When registering a user, the password also has a minimum requirement of length 6
